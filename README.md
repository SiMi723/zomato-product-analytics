# 🍽️ Zomato Product Analytics Dashboard

An end-to-end Product Analytics project built using **SQL**, **Python**, and **Power BI** to analyze customer ordering behavior, restaurant performance, and key business metrics. The dashboard provides actionable insights that can support data-driven product and business decisions.

---

## 📌 Project Overview

Online food delivery platforms generate large volumes of transactional data. Understanding customer behavior, restaurant performance, and ordering trends is essential for improving user experience and business growth.

This project analyzes a Zomato dataset to answer key business questions and presents the findings through an interactive Power BI dashboard.

---

## 🎯 Objectives

- Analyze customer ordering behavior
- Study restaurant performance across cities
- Calculate business KPIs
- Discover ordering trends
- Build an interactive dashboard for business decision-making

---

## 📊 Key Performance Indicators (KPIs)

- Total Orders
- Total Revenue
- Average Order Value (AOV)
- Average Restaurant Rating
- Monthly Order Trends
- City-wise Sales
- Top Restaurants
- Category-wise Revenue
